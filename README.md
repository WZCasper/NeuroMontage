# 🎮 AutoMontage v2 — Автоматический монтаж хайлайтов

Бесплатное веб-приложение для автоматической нарезки игровых хайлайтов.
Работает **полностью в браузере** — видео никуда не загружается, никаких серверов.

---

## ✨ Что умеет v2

| Функция | Описание |
|---|---|
| 🔍 FFT-анализ звука | Разделяет частотные полосы: выстрелы (высокие), взрывы (суббас), голос/смех (средние) |
| 📈 Интерактивная аудиограмма | Видишь волну и все найденные моменты с цветовой маркировкой |
| 🎞️ Превью каждого клипа | Нажми на карточку — откроется мини-плеер с нужным фрагментом |
| ✅ Редактор сегментов | Включай/выключай отдельные клипы перед финальным рендером |
| 🤖 AI-анализ кадров | Claude смотрит на кадры: килл-фид, урон, взрывы (опционально) |
| ⬇️ Скачать MP4 | Готовая нарезка одним файлом |

## 🎬 Рабочий процесс

```
1. Загрузи видео (MP4/AVI/MOV/MKV/WebM до 4 ГБ)
2. Настрой чувствительность и типы моментов
3. Нажми [АНАЛИЗИРОВАТЬ]
       ↓
   FFmpeg.wasm — извлекает аудио
   FFT-анализ — находит пики по полосам частот:
     🔫 Выстрелы → высокочастотные транзиенты
     💥 Взрывы   → суббасовые пики
     😱 Реакции  → голосовая полоса (400–3500 Гц)
     😂 Смех     → устойчивая активность голосовой полосы
4. Смотри аудиограмму с отмеченными моментами
5. Нажимай на карточки — предпросматривай клипы
6. Исключи неудачные → [Склеить выбранные]
7. Скачай готовый MP4
```

## ⚙️ Настройки

| Параметр | Что делает |
|---|---|
| Чувствительность | Больше % → ниже порог → больше клипов |
| Мин. длина клипа | Слишком короткие клипы удаляются |
| Макс. длина клипа | Длинные эпизоды обрезаются |
| Буфер ДО | Секунд до найденного момента |
| Буфер ПОСЛЕ | Секунд после момента (реакция стримера) |
